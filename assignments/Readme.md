\# Internship Assignments



This subdirectory contains all the assignments completed during my internship.



Each assignment is organized and labeled sequentially as:



A01, A02, A03, ...



\## Structure



For every assignment folder, the following documents are included:



\* \*\*Project Guidelines\*\* – The instructions and requirements provided for the assignment.

\* \*\*Project Deliverable\*\* – The completed work submitted for the assignment.



\## File Naming Convention



Each assignment follows this naming format:



\* `A01 Project Guidelines.pdf`

\* `A01 Project Deliverable.pdf`



Similarly:



\* `A02 Project Guidelines.pdf`

\* `A02 Project Deliverable.pdf`

\* `A03 Project Guidelines.pdf`

\* `A03 Project Deliverable.pdf`

\* and so on.



This structure ensures clarity, consistency, and easy navigation across all internship assignments.
